About Minecraft Force Op

Minecraft Force op is a light weight web application that attepts to exploit a recently discovered bug in the Minecraft server source code. This application silently ops your player, meaning that when you become op, admins do not know unless if they view the console log. Commands made by your player are also suppressed, so even if an administrator is online, you are still safe from getting caught! Remember to be responsable with this application.
